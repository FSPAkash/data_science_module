Interactive: Simple Linear Regression

How to use this chart
  • Click anywhere on the grey area below to place a data point. Each click adds one observation with an X (temperature) and Y (yield) value.
  • After placing 2 or more points, a regression line will appear automatically, computed using ordinary least squares (OLS).
  • The dashed purple lines show residuals, the vertical distance between each point and the regression line. OLS minimizes the sum of these squared distances.
  • Watch how R-squared changes as you add points. Try placing points in a tight line (high R-squared) vs. scattered (low R-squared).
  • Try adding an outlier far from the cluster to see how a single extreme point can pull the entire regression line.
